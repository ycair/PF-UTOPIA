PF UTOPIA

安逸烏托邦 · Discord MMORPG

🗺️ 遊戲簡介

PF UTOPIA 是一款運行於 Discord 伺服器內的文字 MMORPG。14 個節點的圖論世界地圖、5 種貨幣經濟體系、4 個副本 Boss、6 種投資股票、寵物抽卡養成、離線打坐掛機——全部透過 Slash 指令即時互動。

你的角色從烏托邦主城出發，穿越初始草原、翡翠森林、沿海小徑，挑戰舊城邦的輻射殭屍與山林後花園的女僕 Boss。在投資交易所低買高賣烏龜股票，在彩券中心試手氣，在大士爺廟膜拜獲得鬼王加持。

🔬 AI Agent 適配性實驗

本專案不僅是一款 Discord 遊戲，更是一項針對「大語言模型驅動之 AI Agent 在網路拓樸與動態經濟環境下之適配性與決策行為」的學術與技術實驗。

📡 資料收集：玩家在圖論地圖上的移動軌跡、供需演算法下的交易決策、戰鬥策略選擇，皆為研究 AI 代理在複雜遊戲環境中行為模式的關鍵觀測數據。

🔒 去識別化：所有對外發表的實驗數據均經過嚴格去識別化處理，不包含任何可直接識別個人身分的資訊。

🤖 未來 AI GM：專案架構已預留 AI Game Master 介面，未來將由 LLM 動態生成劇情、觸發隨機事件、管理世界狀態。
