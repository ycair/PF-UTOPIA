# PF UTOPIA — 安逸烏托邦

Discord 伺服器內運行的 MMORPG 文字遊戲。開源此專案讓玩家了解遊戲機制、數值公式與完整系統設計。

---

## 🌍 地圖與移動

15 個節點的圖論地圖，城內安全區與城外危險區分明：

```
          [世界魔皇巢穴] -- [舊城邦] ☢️ 輻射    [大士爺廟]
                 \             |              /
                  \            |             /
                   +--- [翡翠森林] ----+
                                   |
          [山林後的花園]            |            [寵物天堂]
                 |                  |                |
          [搗蛋精靈之森] --- [初始草原] --- [沿海小徑]
                                   |
                            [🏰 主城] ← 城牆保護
                            /   |   \    \        \
                    [競技場] [彩券] [投資] [女僕教堂] [冒險者公會]
```

- **城內**：主城 + 競技場、彩券中心、投資交易所、女僕教堂、冒險者公會 — 安全，無野怪
- **城外**：初始草原為唯一閘道，出城後隨時可能遇敵，血量跨戰鬥持久不重置
- **特殊節點**：舊城邦（☢️ 輻射 -50% 防禦）、大士爺廟（鬼王庇佑安全區）
- `/move <節點>` 移動，`/travel_status` 查看狀態

## 💀 血量與死亡

血量跨戰鬥**持久保留**。三種回血方式：

| 方式 | 效果 |
|------|------|
| 🏰 回到主城 | 自動滿血 |
| 🙏 大士爺廟膜拜 | 直接滿血 |
| 🧃 恢復劑 / 強效恢復劑 | +50 / +150 HP |

陣亡後使用 `/revive` 花費 **1,500 托幣**復活，出生主城滿血。

## 🌈 世界魔皇 — 全服沖榜

**幻彩暴走・彩虹羊** HP 5,500,000（全服共享），每人每日 2 次，不耗體力。

| 機制 | 說明 |
|------|------|
| 🛡️ 七彩羊毛護體 | 每回合防禦 +5%（最高傷害 80,000） |
| 🌈 虹光共振 | 敵方傷害 +20% |
| ⏱️ 暴走回合 | 35 回合強制結束 |
| 🔄 重置 | 管理員手動重置 |

| 獎勵 | 條件 |
|------|------|
| 👑 擊殺者 | +30 逸幣 |
| 🥇 第 1 名 | +400 烏幣 |
| 🥈🥉 第 2~3 名 | +200 烏幣 |
| 🎯 里程碑 | 10K→50 / 25K→50 / 50K→100 / 100K→200 烏幣 |

`/world_boss` 討伐 | `/world_boss_rank` 排行榜

## 🎟 托幣彩券

- 01~20 選 3 個號碼，開 3+1 特別號
- 頭獎 100 倍 / 貳獎（2+特別號）10 倍 / 叁獎（2 碼）5 倍
- 購買時 DM 確認彩票

## 🛐 信仰

### ⛪ 女僕教堂 — 離線打坐掛機

`/meditate_start` 開始打坐 → 離線累積獎勵 → `/meditate_stop` 領取：
- 💴 托幣（每小時 ~60 元）
- 📦 初始草原戰利品隨機掉落
- ✨ 元神經驗（提升元神等級 → HP+5、體力上限+5）
- ⚠️ 必須位於女僕教堂節點；打坐期間體力不回復
- 🌲 翡翠森林探索有 15% 機率迷路深入內部

### 🏯 大士爺廟 — 膜拜

供奉台灣民間信仰的「大士爺」（面燃大士／普渡公），觀音化現的鬼王。

`/pray` 每日 3 柱香，一次消耗 3 柱：
- ❤️ 滿血恢復
- ⚔️ 攻擊力 +50%（30 分鐘）
- 📈 修為累積（提升修為等級 → 能力點+1）

## 角色屬性

| 屬性 | 初始 | 說明 |
|------|------|------|
| 攻擊力 | 10 | 影響傷害 |

| 血量 | 100 | 持久不重置 |
