# PF UTOPIA — 安逸烏托邦

Discord 伺服器內運行的 MMORPG 文字遊戲。開源此專案讓玩家了解遊戲機制、數值公式與完整系統設計。

---

## 🌍 地圖與移動

14 個節點的圖論地圖，城內安全區與城外危險區分明：

```
     [世界魔皇巢穴]    [舊城邦] ☢️ 輻射    [大士爺廟]
                \              |              /
                 \             |             /
                  +---- [翡翠森林] ----+
                                   |
          [山林後的花園]            |            [寵物天堂]
                 |                  |                |
          [搗蛋精靈之森] --- [初始草原] --- [沿海小徑]
                                   |
                            [🏰 主城] ← 城牆保護
                           /   |   \    \
                   [競技場] [彩券] [投資] [女僕教堂]
```

- **城內**：主城 + 競技場、彩券中心、投資交易所、女僕教堂 — 安全，無野怪
- **城外**：初始草原為唯一閘道，出城後隨時可能遇敵，血量跨戰鬥持久不重置
- **特殊節點**：舊城邦（☢️ 輻射 -50% 防禦）、大士爺廟（鬼王庇佑安全區）
- `/move <節點>` 移動，`/travel_status` 查看狀態

## 💀 血量與死亡

血量跨戰鬥**持久保留**。三種回血方式：

| 方式 | 效果 |
|------|------|
| 🏰 回到主城 | 自動滿血 |
| 🙏 大士爺廟膜拜 | 直接滿血 |
| 🧃 恢復劑 / 強效恢復劑 | +50 / +150 HP |

陣亡後使用 `/revive` 花費 **1,500 托幣**復活，出生主城滿血。

## ⚔️ 戰鬥

`/explore` 和 `/dungeon` 自動依玩家所在節點判定可戰鬥的怪物與 Boss。

**5 個野外節點**：

| 節點 | Boss | 怪物 |
|------|------|------|
| 初始草原 | — | 史萊姆、野雞、兔兔、綿羊、鹿、狼、野豬 |
| 翡翠森林 | — | 猴子、蛇、哥布林、大蜥蜴、食人魔 |
| 沿海小徑 | — | 巨蟹、海妖 |
| 搗蛋精靈之森 | — | 搗蛋精靈（初/中/高） |
| 寵物天堂 | — | 野生寵物 |

**Boss 節點**：

| 節點 | Boss | 建議戰力 | 完美戰力 |
|------|------|---------|---------|
| 🏚️ 舊城邦 ☢️ | 征服入侵殭屍 | 2,800 | 3,000 |
| 🏚️ 舊城邦 ☢️ | 征服入侵軍團 | 24,300 | 27,000 |
| 🌸 山林後的花園 | 解開女僕的緞帶 | 218,700 | 243,000 |
| 🌸 山林後的花園 | 解開女僕們的緞帶 | 1,749,600 | 2,187,000 |

**世界魔皇**：彩虹羊 HP 600,000，位於世界魔皇巢穴。

**傷害公式**：`max(1, (攻擊力 × buff − 敵方防禦 × 0.5) × 隨機(0.8~1.2))`

**能力評分**：`攻擊 × 100 + 防禦 × 50 + 血量 × 10`

## 💰 經濟

### 5 種貨幣

```
簽到拜拜 ──→ 🪙 安幣（活躍幣） ──→ 能力重置
贊助贊助 ──→ 💵 逸幣（贊助幣） ──→ 恢復體力
活動得獎 ──→ 💶 烏幣（活動幣） ──→ 提升戰力、體力上限
賣戰利品 ──→ 💴 托幣（副本幣） ──→ 買道具、彩券、股票
尚未安排 ──→ 💷 邦幣（公共財） ──→ 世界建設（規劃中）
```

| 貨幣 | 取得 | 用途 |
|------|------|------|
| 🪙 安幣 | 每日簽到（唯一來源） | 能力重置 |
| 💵 逸幣 | BMC 儲值 | 恢復體力 |
| 💶 烏幣 | 世界魔皇討伐 | 提升戰力、體力上限 |
| 💴 托幣 | 出售戰利品、打坐掛機、世界魔皇 | 買道具、彩券、股票 |
| 💷 邦幣 | 全服目標（規劃中） | 世界建設（規劃中） |

### 商店

- 節點差異化收購價：主城 100%，野外僅收本地掉落物，最高 30% 主城價
- 消耗品用托幣購買，固定價格
- `/shop_price` 查詢當前所在節點的賣價

## 🎟 托幣彩券

- 01~20 選 3 個號碼，開 3+1 特別號
- 頭獎 100 倍 / 貳獎（2+特別號）10 倍 / 叁獎（2 碼）5 倍

## 🛐 信仰

### ⛪ 女僕教堂 — 離線打坐掛機

`/meditate_start` 開始打坐 → 離線累積獎勵 → `/meditate_stop` 領取：
- 💴 托幣（每小時 ~60 元）
- 📦 初始草原戰利品隨機掉落
- ✨ 元神經驗（提升元神等級 → HP+5、體力上限+5）

### 🏯 大士爺廟 — 膜拜

供奉台灣民間信仰的「大士爺」（面燃大士／普渡公），觀音化現的鬼王。

`/pray` 每日 3 柱香，一次消耗 3 柱：
- ❤️ 滿血恢復
- ⚔️ 攻擊力 +50%（30 分鐘）
- 📈 修為累積（提升修為等級 → 能力點+1）
